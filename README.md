# Food Ranker
Interface that can taker user input to find the best food for their specified category from a large database.

In order to run, make sure to have PyGame installed. You can ensure so by typing one of two commands into your command line:

1. `pip install pygame`

2. `python -m pip install pygame`

After that, ensure that you are in the overall food_ranker directory (being within any subdirectory will cause issues with imports).
You should then be able to run the following command:

`python -m interface.user_interface`

After this, a window should pop up with the Food Ranker application ready to be used.
Enjoy!
